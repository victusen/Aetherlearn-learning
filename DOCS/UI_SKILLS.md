# UI Design Principles & Skills

This document summarizes the core UI/UX design principles and skills gathered from the curated resources in the `design_resources` folder.

## 1. Core Design Principles
These are fundamental guidelines for creating effective user interfaces:

- **Hierarchy**: Use size, color, and spacing to guide the user's eye to the most important elements first.
- **Contrast**: Ensure text and interactive elements stand out from the background for readability and focus.
- **Consistency**: Maintain a uniform look and feel (buttons, colors, typography) across the entire application to reduce cognitive load.
- **Alignment**: Align elements to a grid or relative to each other to create a sense of order and professionalism.
- **Proximity**: Group related items together to show their relationship.
- **Balance**: Distribute visual weight evenly to create a harmonious layout.

## 2. Modern Design Techniques
- **Glassmorphism**: Using frosted-glass effects with background blur.
- **Micro-interactions**: Subtle animations that provide feedback (e.g., button hover, loading states).
- **Dark Mode**: Designing for low-light environments with high-contrast, eye-friendly palettes.
- **Responsive Design**: Ensuring layouts work seamlessly from mobile to desktop.

## 3. Recommended Skillsets for 2024-2025
- **Design Systems**: Learning how to use and build component libraries (e.g., Radix UI, Shadcn UI).
- **Accessibility (a11y)**: Ensuring interfaces are usable by everyone, including those using screen readers.
- **Typography**: Choosing and pairing fonts that communicate the brand's voice and improve legibility.
- **Prototyping**: Using tools like Figma to test flows before implementation.
